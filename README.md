## ✨ Features

- **🎯 Level-Based Learning**: Queries organized into 3 levels - Beginner, Intermediate, and Advanced
- **🔍 Advanced Search**: Quick search through all queries and concepts
- **🎨 Modern UI**: Beautiful Glassmorphism design with Tailwind CSS
- **📱 Fully Responsive**: Works perfectly on all devices (mobile, tablet, desktop)
- **📋 Copy to Clipboard**: One-click copy functionality for all code examples
- **🎯 Filter System**: Filter queries by difficulty level
- **💡 Complete Explanations**: Each query includes detailed explanation, examples, and tips
- **⚡ Interactive**: Click on any query card to expand and see full details
- **🌙 Dark Theme**: Eye-friendly dark mode design

## 📊 Query Categories

### 🟢 Beginner Level (9 Concepts)
| Query | Description |
|-------|-------------|
| **SELECT** | Retrieve and display data from tables |
| **WHERE** | Filter records based on conditions |
| **COUNT** | Count the number of records |
| **SUM** | Calculate sum of numeric values |
| **AVG** | Calculate average of numeric values |
| **MIN** | Find the minimum value |
| **MAX** | Find the maximum value |
| **DELETE** | Delete records from a table |
| **UPDATE** | Update existing records |

### 🟡 Intermediate Level (3 Concepts)
| Query | Description |
|-------|-------------|
| **RANDOM** | Select random records |
| **PRIMARY KEY** | Unique identifier for each record |
| **FOREIGN KEY** | Link tables together |

### 🔴 Advanced Concepts (2 Concepts)
| Concept | Description |
|---------|-------------|
| **One-to-One** | One-to-One relationship between tables |
| **One-to-Many** | One-to-Many relationship between tables |
